# Oluwaseun's Data Analysis Portfolio

Welcome to my data analysis portfolio website! This is a personal project showcase built with HTML, CSS, and JavaScript.

## 🌐 Website

Visit my portfolio at: [Oluwaseun-Macfoy.github.io](https://Oluwaseun-Macfoy.github.io)

## 📁 Project Structure

```
├── index.html       # Main portfolio page
├── styles.css       # Styling
├── script.js        # Interactive features
└── README.md        # This file
```

## ✨ Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Navigation**: Easy navigation between sections (Home, About, Projects, Skills, Contact)
- **Project Showcase**: Display your data analysis projects with descriptions and tools used
- **Skills Section**: Highlight your programming languages, tools, and other skills
- **Contact Links**: Quick links to GitHub, LinkedIn, and email
- **Smooth Animations**: Elegant transitions and hover effects

## 🚀 Getting Started

### To customize your portfolio:

1. **Update Contact Information**
   - Edit the Contact section in `index.html`
   - Replace placeholder email, LinkedIn, and GitHub URLs

2. **Add Your Projects**
   - Update project titles, descriptions, and tags in `index.html`
   - Add links to your project repositories or detailed project pages

3. **Update Skills**
   - Modify the skills in the Skills & Tools section
   - Add your proficiencies and technical expertise

4. **Personalize About Section**
   - Update the About section with your background and experience
   - Make it reflect your unique story

5. **Adjust Colors** (Optional)
   - Edit the color variables in `styles.css`
   - Primary color: `--primary-color`
   - Accent color: `--accent-color`

## 📝 Content Tips

- Keep project descriptions concise but informative
- Include relevant tags/tools for each project
- Link to actual project repositories or dashboards
- Add data visualization screenshots if possible
- Highlight key achievements and metrics

## 🔄 Deployment

This site is automatically deployed to GitHub Pages. Any changes pushed to the `main` branch will be live within minutes!

To push changes:
```bash
git add .
git commit -m "Update portfolio content"
git push origin main
```
